Cubyz at its core heavily relies on open-source community contributions, so any help you can provide is greatly appreciated.

---
## Code & Development
If you are interested in helping by providing code or creating addons, you can check these resources:

* **Getting Started:** Review the [Cubyz Contributions Guide](https://github.com/PixelGuys/Cubyz/blob/master/CONTRIBUTING.md) on GitHub for code standards and repository rules.
* **Issue Tracking:** Browse the [GitHub Issues](https://github.com/PixelGuys/Cubyz/issues) page to find open bugs, feature requests, or tasks labeled contributor friendly.
* **Addon Development:** Create addons that the community will love playing with and enjoy!
---
## Community Volunteering
You don't need to know how to code to make an impact. Keeping the community active, welcoming, and well-documented is just as vital to the growth of Cubyz.

* **Youtube Team:** Help create update videos, development blogs, and more for the [Official Cubyz Youtube Channel](https://www.youtube.com/@CubyzGame). To get involved with managing and producing content for the channel, you can fill out the [YouTube Team Form](https://forms.gle/vKpye6XXjUXyVYsz5).
* **Discord Team:** Help keep the community active, safe, and engaging by managing text channels or helping out with events on the official Cubyz Discord. To apply for a moderation role, you can use the [Discord Team Form](https://forms.gle/5jDWA6ELoivp1KzA9).
* **Wiki Team:** Help keep the wiki up-to-date and improve documentation to help new players get started. Anyone can contribute directly by making pull requests to the [Cubyz Wiki Repository](https://github.com/PixelGuys/Cubyz-Wiki). If you want to take on a dedicated role maintaining and organizing the wiki pages, you can apply via the [Wiki Team Form](https://forms.gle/Zu4bFgL4MoqEUJcGA).
* **Github Team:** Help keep the project organized by reviewing open pull requests and triaging issues on the [Cubyz GitHub](https://github.com/PixelGuys/Cubyz). While anyone can submit code and fixes at any time, you can apply for a more dedicated role using the [Github Team Form](https://forms.gle/pqiEjZZNfcnrrKMY7).
* **Reddit Team:** Keep community spaces safe, organized, and active by moderating the official [Cubyz Reddit](https://www.reddit.com/r/CubyzGame/). To apply for a moderation role on the subreddit, check out the [Reddit Team Form](https://forms.gle/Zg4oizr48UJY6yro9).

## Donations
If you love the project but don't have the time to contribute code or community hours, you can support development via a donation. You can choose exactly who or what your funds support from the options below.

* **Project Lead**: If you want to support the core game mechanics, engine updates, and overall project management, you can back the project leader [Quantum Developer](https://github.com/IntegratedQuantum), via the Cubyz Patreon.[Cubyz Patreon](https://www.patreon.com/cw/Cubyz).

* **Art & Visuals**: If you love the game's art direction, custom textures, and 3D models / designs, you can back our artist [Carrie](https://github.com/careeoki), via [Ko-Fi](https://ko-fi.com/careeoki)
