# Cubyz Wiki

## About Cubyz

**Cubyz** is a 3D voxel sandbox game inspired by Minecraft, featuring innovative mechanics and limitless possibilities.

---

## Unique Features

<div class="grid cards" markdown>

-   :material-earth-box:{ .lg .middle } Level of Detail

    Play with huge render distances without the huge performance cost.

    See up to 3000 blocks ahead of you!

- :material-package:{ .lg .middle } 3D Chunks

    Infinite terrain in all directions! Build without limits or explore endless caves.

    Explore the world horizontally and vertically, 1 million+ blocks in every direction!

- :material-hammer-screwdriver:{ .lg .middle } Procedural Crafting

    Create your own tools by combining a variety of unique materials!

    Discover effects, interactions and find the best tool for the job!
